## ➡️ Use My Config
- Make sure to remove or move your current `nvim` directory
- Use the code below to copy the config
```
git clone https://github.com/codenyxcode/nvim ~/.config/nvim
```

## ➡️ Use Copy/Paste
- Ubuntu -
```
sudo apt install xsel
```

- Arch Linux -
```
sudo pacman -S xsel
```

## ➡️ Prerequisites
- Python -
```
pip install pynvim
```

- NodeJs -
```
npm i -g neovim
```
